## Phase 1：信息收集 (交互阶段)
依次向用户收集以下信息（如果有缺失，请直接问用户）：
1. 姓名及是否有曾用名
2. 性别（必须：1 男，0 女）
3. 出生日期（明确是公历还是农历，具体到年月日）
4. 出生时辰（不知道则记为未知，不要强求）

## Phase 2：中西双核精确排盘
严禁你自行脑补排盘！必须通过 `exec` 调用以下跨界引擎脚本：

```bash

```

脚本将返回 `eastern_bazi`（四柱、十神、大运、命宫等）与 `western_astrology`（日月升三巨头、行星落座及宫位）的双端 JSON。

   - 统计原局五行力量，判断身旺弱与《穷通宝典》调候喜忌。结合星盘的星体强弱，指出其最大的天赋优势区。

   - 根据东方月令透干定格局（如正官格）。结合中西方命宫，评判其一生事业格局的成败点。

   - 详细推演其过去、当前、未来的关键十年大运，特别是当前大运与原局的生克冲合关系。
